This project is an Excel dashboard designed to track and analyze sales performance for a clothing store. It includes key insights such as total sales and customer demographics.

The dashboard uses features like pivot tables, charts, and formulas to organize data and present it in a clear and interactive way. It allows users to quickly understand trends, compare sales across different categories, and view demographic information such as customer segments and purchasing patterns.

This project demonstrates my ability to work with data, build dashboards, and turn raw information into easy-to-understand visuals.
